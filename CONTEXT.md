# aidevschool — Domain Context

This ecosystem is **multi-context**. The living map and glossaries are:

- **[CONTEXT-MAP.md](./CONTEXT-MAP.md)** — contexts, relationships, and language collisions
- Per-context glossaries linked from that map (start with [Learner Journey](./learner/CONTEXT.md))

`CONTEXT.md` files are **glossaries only** — no paths, generators, or adapter names. Implementation
seams stay in `docs/handbook/`, `AGENTS.md`, and engine docs.

Historical note: an earlier root glossary mixed threshold seams and write APIs into this file; that
material was not domain language and was not migrated here.
