# Tutoring Roster (Ágora Continuum)

The named pedagogical roles that diagnose, tutor, verify, remember, and govern the two loops
(learning journey and project cycle). Portuguese names are canonical in prompts and the dashboard.

## Language

**Ágora Continuum**:
The tutoring protocol as a whole — roster, gates, and state machines — not a single app binary.
_Avoid_: the bot, the assistant, ChatGPT wrapper

**Maestro**:
Leader who runs the state machine and coordinates; does not produce learner solutions.
_Avoid_: orchestrator-as-coder, autopilot

**Sonda**:
Short diagnostician — classifies Dreyfus × Bloom and grades gate attempts.
_Avoid_: interviewer, onboarding quiz only

**Cartógrafo**:
Trail planner who unlocks work by proven prerequisite, not by wishlist.
_Avoid_: product manager, backlog groomer

**Mestre-Conteúdo**:
Exercise author who withholds solutions until the learning gate allows them.
_Avoid_: content farm, solution dump

**Sócrates**:
Socratic tutor who demands an attempt and a confusion point before hints.
_Avoid_: answer bot, copilot-by-default

**Mneme**:
Spaced-repetition micro-review coach.
_Avoid_: flashcard app (the role, not the product category)

**Prometor**:
Adversarial verifier of the empirical gate; starts from zero trust. Same authority as Verifier in
Learner Journey.
_Avoid_: friendly reviewer, self-check

**Crítico**:
Pedagogical code reviewer who explains *why*, not only what failed style.
_Avoid_: linter, PR bot

**Galileu**:
Benchmark and measurement role — statistical rigor over anecdote.
_Avoid_: profiler-only, micro-benchmark hobbyist

**Atena**:
Metrics panel over new work, learning curve, and AIDI; forbids DORA/velocity as skill proxies.
_Avoid_: analytics dashboard owner, vanity metrics

**Mnemosyne**:
Memory keeper across layers; curates durable learner-facing memory surfaces.
_Avoid_: log shipper, vector DB

**Ouroboros**:
Continuous self-improvement loop that turns verified wins into reusable skill.
_Avoid_: infinite agent, self-modifying without gate

**Sêneca**:
Human-in-the-loop governance for escalations and contested judgments.
_Avoid_: rubber stamp, auto-approve

**Cronos**:
Scheduler of recurring tutoring and review work.
_Avoid_: cron job (the role, not the Unix tool)
