# Teaching Game — Voxel

Turns one curriculum Concept into a 3D simulation attempt. Same contract as Pixel: emit evidence,
never mark mastery. Genre is structure and dynamics, not arcade budget pressure.

## Shared teaching-game language

**Teaching Game**:
A playable attempt surface for exactly one Concept from one Project and one learning Unit.
_Avoid_: demo, visualization-only (if it cannot emit Evidence Records)

**Evidence Record**:
One immutable structured record of a cleared or failed play segment. Raw input to the Verifier.
_Avoid_: screenshot as sole proof, editor scene file

**Scenario**:
A designed simulation situation the learner operates to reveal the Concept.
_Avoid_: Encounter (pixel term), level pack without a concept

## Voxel-specific language

**Sim**:
The running model of the system under study inside the Scenario.
_Avoid_: game world (too generic), digital twin (overclaim)

**Scene**:
The rendered 3D projection of the Sim for the learner.
_Avoid_: level mesh, map only

**Station**:
A locus in the Scenario where the learner acts or observes a responsibility.
_Avoid_: NPC, checkpoint

**Ring**:
A circular topology used when the Concept is consistent hashing or similar placement.
_Avoid_: circle UI decoration, progress ring
