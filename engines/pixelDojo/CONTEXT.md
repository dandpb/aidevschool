# Teaching Game — Pixel

Turns one curriculum Concept into a 2D arcade attempt. The game is an attempt surface: it emits
raw evidence and never writes mastery.

Shared rules with Voxel live in the teaching-game contract; this glossary is the pixel genre.

## Shared teaching-game language

**Teaching Game**:
A playable attempt surface for exactly one Concept from one Project and one learning Unit.
_Avoid_: demo, toy, portfolio piece

**Evidence Record**:
One immutable structured record of a cleared or failed play segment. Raw input to the Verifier;
not itself Mastered.
_Avoid_: save file, high score, localStorage progress

**Encounter**:
A discrete play challenge inside the pixel game that teaches through a mechanic.
_Avoid_: level (prefer when you mean progression index), quest (broader narrative)

## Pixel-specific language

**Wave**:
A timed or sequenced pressure segment inside an Encounter.
_Avoid_: round, boss fight (unless the mechanic truly is one)

**Heat**:
A budget/pressure meter that rises under load and punishes overuse of capacity.
_Avoid_: health, mana (wrong metaphor unless the concept is literally that)

**Overheat**:
The failure state when Heat exceeds the allowed budget.
_Avoid_: game over (terminal framing), crash
