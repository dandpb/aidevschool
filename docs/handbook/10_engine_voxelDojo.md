# Engine — voxelDojo

The 3D teaching-simulation engine: each game is a Three.js simulation that turns one
`curriculum/` concept into an operable 3D system and emits evidence for the separate verifier.
Sibling of pixelDojo — same [teaching-game contract](../design/teaching-game-contract.md),
different genre: pixelDojo gamifies **rules and budgets**; voxelDojo makes **structures and
dynamics** (rings, topologies, quorums, flows) tangible.

## The pilot — Game 10 "HASH RING"

Teaches **consistent hashing** (project `10_distributed_cache`, unit `U9-distributed-cache`) as an
orbital ring of cache stations. L1 predict-the-owner, L2 join/leave arc prediction, L3 fix load
skew with virtual nodes, L4 the modulo-storm contrast. The sim core is headless and Vitest-proven
(K/N moved bound, vnode skew reduction, modulo ≈ (1−1/N) reshuffle); the Three.js scene only
renders sim state; a Playwright smoke plays L1–L2 in a real browser and asserts `EVIDENCE` records.

```bash
cd engines/voxelDojo/game-10-hash-ring
pnpm install
pnpm run dev         # play at localhost:5173
pnpm run test        # Vitest — concept proofs + controller playthroughs
pnpm run typecheck   # tsc --noEmit
pnpm run smoke       # Playwright — browser playthrough + evidence assertion
```

## Contracts to respect

- Evidence: `source: "voxeldojo"`, emitted via `window.__voxelDojoEvidence` + `EVIDENCE <json>`
  console records. Shape and rules in the [teaching-game contract](../design/teaching-game-contract.md).
- Scheduling: read-only review slice generated by the substrate into
  `game-10-hash-ring/src/content/reviewSlice.ts` (`python3 -m learner.substrate` regenerates it).
  Whether an attempt gates first mastery or counts as review is decided by learner state: as of
  2026-07-05 `U9-distributed-cache` is not yet mastered, so HASH RING evidence can serve the real
  learning gate; the emitter derives `review_context` from the slice either way.
- The game never writes learner state; `python3 -m engines.pixelDojo.verifier` (source-agnostic)
  owns gate decisions.
