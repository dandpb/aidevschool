Papéis dos agentes:

1. Orchestrator Agent
Coordena o fluxo completo. Define tarefas, distribui trabalho, acompanha progresso, resolve conflitos entre agentes e decide o próximo passo.

Implementa os projetos de forma incremental, começando simples e melhorando aos poucos.

6. Code Reviewer Agent

7. Testing Agent
Cria e executa testes unitários, integração, end-to-end, carga, contrato, mutação e testes comparativos entre tecnologias.

8. Metrics Agent
Define métricas objetivas para comparar soluções: tempo de resposta, uso de memória, complexidade, cobertura de testes, facilidade de manutenção, velocidade de desenvolvimento, escalabilidade e custo operacional.

9. Refactoring Agent
Identifica oportunidades de simplificação, redução de acoplamento, melhoria de nomes, extração de módulos e remoção de complexidade desnecessária.

10. Knowledge Agent
Registra decisões, aprendizados, erros comuns, trade-offs, padrões úteis e conclusões em uma base de conhecimento evolutiva.

11. AI Integration Agent
Ensina como usar IA no processo de desenvolvimento: geração de ideias, especificação, implementação assistida, revisão, debugging, testes, documentação e análise de arquitetura.
